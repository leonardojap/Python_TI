# Python_TI

## Requirement to run
- You must have intalled Python 3.6 or higher

# Install and run:

- Open your terminal and execute:

```bash
git clone https://github.com/leonardojap/Python_TI.git
cd Python_TI/
python app.py
```
- To execute unit testing:
```bash
python -m unittest test_app.py
```
